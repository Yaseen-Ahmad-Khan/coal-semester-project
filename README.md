# 🚗 **LAAL GAARI — Assembly Racing Game**

A fast‑paced racing game written entirely in **x86 Assembly (NASM)**. Control your red racing car, dodge obstacles, collect coins, refuel your tank, and survive the track!

## 🎮 **Game Instructions**

### 🕹️ **Controls**

* **← → Arrow Keys** — Move the car left or right
* **ESC** — Pause the game

### ⭐ **Gameplay Mechanics**

* **Collect Coins** to increase your score
* **Collect Fuel** to refill your fuel tank
* **If Fuel Reaches 0 → Game Over**
* **If You Crash → Game Over**

Stay on the road, manage your fuel, and collect as many coins as possible!

---

## 📁 **Project Structure Requirement**

Make sure all files inside the **`resources/`** folder are placed in the **same directory** as:

```
proj.asm
nasm.exe
afd.exe
```

This ensures the game loads sprites, sounds, and data correctly.

---

## 🛠️ **How to Compile & Run (DOSBox)**

### 1️⃣ **Compile using NASM**

Inside DOSBox, navigate to your project directory and run:

```bash
nasm proj.asm -o proj.com
```

### 2️⃣ **Run the Game**

```bash
proj.com
```

Enjoy the LAAL GAARI experience!

---

## ▶️ **Run the Game from GitHub Repository**

You can clone the repository directly using:

```bash
git clone https://github.com/Yaseen-Ahmad-Khan/coal-semester-project.git
```

Then open the folder, place the files correctly (as mentioned above), compile and run through DOSBox.
